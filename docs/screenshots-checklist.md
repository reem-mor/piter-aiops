# Screenshots checklist (submission)

Capture 8–10 PNGs at 1920×1080 or full-window.

1. **Home idle** — before Start Alert Stream
2. **Storm playback** — TopBar timer + alert count visible
3. **P1 modal** — red P1 candidate with Analyze / Escalate / Ask Agent
4. **PITER analysis** — structured sections on Home after Analyze
5. **Enrichment** — suspect deployment + similar incidents table in response
6. **Agent chat** — follow-up in dock with same session; Clear chat visible
7. **Past investigations** — History → Past investigations tab with row
8. **Hybrid fallback** — offline KB banner when `PITER_LOCAL_FALLBACK` active
9. **System / tools** — metrics tools panel (optional)
10. **Health** — `GET /api/health?deep=1` or bootstrap showing agent/KB IDs (optional)

Store under `screenshots/` with descriptive names. See also [`screenshots/deployment_validation.md`](screenshots/deployment_validation.md).
